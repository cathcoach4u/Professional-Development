## Adding New Sessions

Each session can have its own page. Create a new file in `sessions/` using the date format `YYYY-MM-DD.md`.

!!! tip "How to add a new session"
    1. Copy `sessions/YYYY-MM-DD.md` and rename it with the session date
    2. Fill in the details
    3. Add a new row to the table above
    4. Update the [Training Register](../../register.md) supervision hours table
